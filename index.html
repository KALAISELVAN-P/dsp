3.1 Theory

The sine wave or sinusoid is a mathematical curve that describes a smooth repetitive oscillation representing an AC signal whose function is given by, y(t) = A sin(ωt + φ)

Where:
A – Amplitude in Volts
ω = 2πf – Angular Frequency in Hertz
f – Frequency in Hertz
t – Time Period in Seconds
φ – Phase Angle in degrees

A cosine wave is a signal waveform with a shape identical to that of a sine wave, except each point on the cosine wave occurs exactly 1/4 cycle earlier than the corresponding point on the sine wave. A cosine wave and its corresponding sine wave have the same frequency, but the cosine wave leads the sine wave by 90 degrees of phase whose function is given by:

cos(x) = sin(x + π/2)

Sawtooth Waveforms are another type of periodic waveform whose shape resembles the teeth of a saw blade. The sawtooth wave contains a fundamental and both odd and even harmonics. Square-wave Waveforms are used extensively in electronic and microelectronic circuits for clock and timing control signals as they are symmetrical waveforms of equal and square duration representing each half of a cycle and nearly all digital logic circuits use square wave waveforms on their input and output gates.

Exponential waveform is a non-repetitive waveform that rises or falls exponentially from some initial value at some initial time whose function is given by, y(t) = e^(at)

Where:
a > 0, the waveform rises without bound
a < 0, the waveform decays to zero.

An impulse voltage is a uni-directional voltage that rises from zero to a maximum value in a very short time and dies away more or less to zero in a comparatively greater time. A ramp function is a function of voltage that increases or decreases linearly with time. The slope of the ramp function is always constant. A Step function makes an instantaneous jump from one steady value to another steady value. A step means an instantaneous change in level.

FFT and IFFT Using DIT and DIF Algorithm

4.3 Program

DIT-FFT

% Fast fourier transform - DIT using in-built function
clc;
clear all;
x=input('Enter the input sequence:');
N=length(x);
y=fft(x,N);
disp('FFT-DIT o/p using in-built fn:');
disp(y);

% FFT-DIT using algorithm
y=bitrevorder(x);
w1=1;
w2=0.707-0.707j;
w3=-1j;
w4=-0.707-0.707j;

%calculation of stage 1
for K=1:2:N
    s1(K)=y(K)+y(K+1);
    s1(K+1)=y(K)-y(K+1);
end
disp('o/p of 1st stage');
disp(s1);

%calculation of stage 2
for K=1:4:N
    s2(K)=s1(K)+s1(K+2);
    s2(K+1)=s1(K+1)+w3*s1(K+3);
    s2(K+2)=s1(K)-s1(K+2);
    s2(K+3)=s1(K+1)-w3*s1(K+3);
end
disp('o/p of 2nd stage');
disp(s2);

%calculation of stage 3
for K=1:8:N
    s3(K)=s2(K)+s2(K+4);
    s3(K+1)=s2(K+1)+w2*s2(K+5);
    s3(K+2)=s2(K+2)+w3*s2(K+6);
    s3(K+3)=s2(K+3)+w4*s2(K+7);
    s3(K+4)=s2(K)-s2(K+4);
    s3(K+5)=s2(K+1)-w2*s2(K+5);
    s3(K+6)=s2(K+2)-w3*s2(K+6);
    s3(K+7)=s2(K+3)-w4*s2(K+7);
end
disp('o/p of 3rd stage :');
disp(s3);

%magnitude and phase plot
disp('magnitude and phase response of FFT-DIT sequence:');
subplot(2,1,1);
stem(20*log10(abs(s3)));
title('Magnitude plot');
xlabel('sample(n)');
ylabel('magnitude');
subplot(2,1,2);
stem(angle(s3));
title('phase plot');
xlabel('sample(n)');
ylabel('phase');

FIR LOW PASS FILTER DESIGN

%FIR LOW PASS FILTER DESIGN
clc;
clear all;
disp('FIR LOW PASS FILTER DESIGN');
N=input('enter the order of the filter:');
wc=input('enter the cut-off frequency:');
n=0:1:N-1;
M=(N-1)/2;
disp('desired filter coefficient:');
for n=0:N-1
    if(n==M)
        hd(n+1)=(wc/pi);
    else
        hd(n+1)=sin(wc*(n-M))./(pi*(n-M));
    end
end
disp(hd);
disp('rectangular window sequence:');
n=0:N-1;
w1(n+1)=1;
disp(w1);
disp('Filter coefficient 1:');
for n=0:N-1
    h1(n+1)=hd(n+1).*w1(n+1);
end
disp(h1);
disp('hanning window sequence:');
n=0:N-1;
w2(n+1)=0.5-0.5*cos(2*n*pi/(N-1));
disp(w2);
disp('Filter coefficient 2:');
for n=0:N-1
    h2(n+1)=hd(n+1).*w2(n+1);
end
disp(h2);

H1=freqz(h1,1,w);

clear; clc; close all;

fs = 360;
t = 0:1/fs:5;
ecg = 1.5*sin(2*pi*1.2*t) + 0.5*sin(2*pi*3*t);
ecg = ecg + 0.8*randn(size(t));

[b, a] = butter(2, [0.5 40]/(fs/2), 'bandpass');
ecg_filt = filtfilt(b, a, ecg);

[~, locs_R] = findpeaks(ecg_filt, 'MinPeakHeight', mean(ecg_filt)+0.5*std(ecg_filt), ...
    'MinPeakDistance', 0.25*fs);

RR_intervals = diff(locs_R)/fs;
HR = 60 ./ RR_intervals;

figure;
plot(t, ecg_filt, 'b'); hold on;
plot(t(locs_R), ecg_filt(locs_R), 'ro', 'MarkerFaceColor', 'r');
xlabel('Time (s)'); ylabel('Amplitude (mV)');
title('ECG Signal with R-peaks Detected');
legend('Filtered ECG', 'R-peaks');
grid on;

fprintf('Number of heart beats detected: %d\n', length(locs_R));

if ~isempty(HR)
    fprintf('Average Heart Rate: %.2f bpm\n', mean(HR));
else
    fprintf('Not enough peaks to compute HR.\n');
end

figure;
plot(HR, 'r-o');
xlabel('Beat Number'); ylabel('Heart Rate (bpm)');
title('Heart Rate Variation');
grid on;
